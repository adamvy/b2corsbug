Demonstration of a CORS caching issue on chrome using backblaze.

  1. Open this page in chrome
  2. Open chrome devtools
  3. Open the network tab
  4. Ensure that the cache is enabled
  5. Refresh this page until you only see one "TEST" image
  6. Observe the error in the console

On firefox you expect to see the same "Test" image twice. On chrome about 50% of the time you will see two images, and 50% of the time you will see one image and an error message
